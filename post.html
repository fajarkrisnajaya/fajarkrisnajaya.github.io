How to make Githubpages with git

Ada beberapa cara untuk membuat githubpages, salah satunya yaitu melalui git.

> Langkah-langkah membuat githubpages dengan git

1. Langkah pertama yang dilakukan yaitu dengan mencari template HTML & CSS. Saya mendapatkan template di website FreeHTML5.co . Saya memilih template dengan tema yang simple, minimalistik, namun tetap berkesan.

2. Setelah kita mendownload templatenya, kita ekstrak filenya. file template saya bernama “profile”, maka saya lakukan ekstraksi terhadap file tersebut ke directory yang sama.

3. Langkah selanjutnya adalah dengan melakukan editing terhadap templatenya seperti mengganti tulisan, judul, stylesheet,dll . Keterangan tentang apa saja yang telah di-edit bisa dilihat di snippet code di bawah.

4. Login github, lalu buka www.github.com/new untuk membuat repositori baru kosongan. Isikan nama repositorinya dan deskripsinya sesuai dengan konteks.

5. Klik kanan ke sebuah direktori, lalu tekan “Git Bash Here” untuk membuka CLI dari github dan menyambungkan working directorynya.

6. Langkah selanjutnya yaitu dengan melakukan cloning terhadap repositori kosong tersebut menggunakan perintah “git clone” . pada dasarnya perintah tersebut melakukan cloning dari repository github ke local dan melakukan inisialisasi.

7. Akan terbuat sebuah folder baru bernama repositori dari github. Jika kita lihat isi folder direktori di bawah, maka akan ada hidden folder bernaam .git . hidden folder ini menandakan bahwa folder tersebut telah dilakukan inisialisasi.

8. Setelah itu kita masuk ke folder (nama repositori) lalu masukkan command “git add . “ . ini berfungsi untuk menambahkan semua file dan directory untuk nantinya dilakukan commit.

9. Setelah itu kita lakukan commit terhadap filenya dengan cara memasukkan command “git commit – m “first commit” . karena ini merupakan commit pertama kita, maka kita masukkan saja message first commit.

10. Kita bisa mengecek apakah semua file sudah dilakukan commit dengan cara menuliskan comman “git status”

11. Kita juga bisa melihat log dari beberapa commit dengan cara menuliskan git log

12. setelah local repository berhasil dicommit, saatnya untuk melakukan push ke github. Langkahnya adalah dengan cara menuliskan git push –u origin (branch). Di kasus saya, branch nya bernama main.

13. Jika berhasil, maka kita bisa melihat hasil dari file yang telah kita commit di server github repository.

14. Langkah selanjutnya adalah membuat githubpages. Di menu repository, klik tombol setting.

15. Jika sudah maka masuk ke Pages - > atur branch -> save. Akan terlihat URL yang nantinya menjadi domain site kita di repositori github.

16. Kita coba masuk ke url tersebut. Dan hasilnya jika berhasil maka akan terlihat websitenya sesuai dengan script yang kita miliki
